# Paperclip (Symfony)

This project was scaffolded with the Symfony skeleton.

Quick start

1. Install PHP (>=8.2), Composer and Git.
2. From project root:

```powershell
cd "C:\Users\pmcfe\Desktop\New folder (2)\paperclip"
composer install
php -S 127.0.0.1:8000 -t public
```

Or use Symfony CLI if installed:

```powershell
symfony serve
```

Development notes

- Application code is under `src/`.
- Configuration is under `config/`.
- Logs and cache are in `var/`.
